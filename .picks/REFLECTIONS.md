# Picks Reflections

Post-game reflection log. Every settled official pick should get a short entry.

Goal: build real pattern recognition over time, not just a W/L scoreboard.

## Format

Each entry answers:

1. What was the stated edge thesis?
2. What actually decided the game?
3. Was the data available to catch it — did the analysis check?
4. Bad bet or bad result?
5. What changes going forward?
